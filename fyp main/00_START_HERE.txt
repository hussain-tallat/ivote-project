🆘 TROUBLESHOOTING

Problem: "Cannot find module 'xyz'"
  Solution: Run "npm install" in that directory

Problem: "MongoDB connection failed"
  Solution: Ensure MongoDB is running:
    • Windows: mongod (or use MongoDB service)
    • Mac: brew services start mongodb-community
    • Cloud: Use MongoDB Atlas connection string

Problem: "SMTP errors" (email not sending)
  Solution: 
    1. Go to https://myaccount.google.com/apppasswords
    2. Create app-specific password
    3. Update SMTP_PASS in .env
    4. Restart backend server

Problem: "Admin won't login"
  Solution:
    1. Admin must be registered first
    2. Check email for OTP verification
    3. Then login requires OTP (2-factor)
    4. If forgot password, use new forgot password flow

Problem: "Dashboard shows loading forever"
  Solution:
    1. Check backend console for API errors
    2. Verify elections exist in database
    3. Check network tab (F12) for API responses
    4. Ensure adminToken is in localStorage
